Build a complete, production-ready website for BAANS INFRA — India's leading bamboo construction and design company. They build luxury bamboo resorts, Bali-style villas, mud & rammed earth constructions, pavilions, cafes, and premium stays across India.

---

## TECH STACK
- React + Vite
- React Router DOM for multi-page routing
- Plain CSS (no Tailwind) — write custom CSS with CSS variables
- No UI libraries. Every component hand-crafted.
- Framer Motion for all scroll animations and transitions
- GSAP for the hero scroll-scrub frame animation
- Use a central `src/data/data.ts` file for ALL site content — nothing hardcoded into pages. Every text, image path, project detail, testimonial, process step, FAQ, nav link — everything lives in data.ts and is imported into components.

## GLOBAL STYLES
- Custom cursor: a small filled circle (12px, --color-primary) that follows the mouse smoothly with a trailing larger ring (28px, semi-transparent). On hover over links/buttons, the inner dot scales up and changes to --color-accent. The cursor should feel organic, not mechanical.
- Smooth scroll: `scroll-behavior: smooth` on html
- All images: `border-radius: 16px` by default, some hero/full-bleed images have `border-radius: 24px`
- Section scroll animations: every card, image, heading fades up with `opacity: 0 → 1` and `translateY: 40px → 0` on entering viewport. Use Framer Motion's `whileInView` with `viewport: { once: true, amount: 0.2 }`. Stagger children by 0.12s.
- No section should feel harsh or technical. Every transition must feel like nature breathing.

- Frame logic: as user scrolls from 0 to 500vh, map scroll progress (0–1) to frame index (1–60). Round to nearest integer. Display the corresponding frame as a full-bleed background image (`object-fit: cover`, `width: 100%`, `height: 100%`).
